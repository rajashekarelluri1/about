SQL Query Order

How SQL is written vs. how it is executed: the key to debugging complex queries.

Writing Order Syntactical

1
SELECT

Pick columns, aliases, & aggregates

2
FROM

Source tables & JOIN conditions

3
WHERE

Filter base rows (pre-aggregation)

4
GROUP BY

Combine rows based on column values

5
HAVING

Filter aggregated groups

6
ORDER BY

Sort the final output

7
LIMIT

Restrict number of rows returned

Execution Order Logical

1
FROM & JOIN

Engine identifies source tables first

2
WHERE

Eliminate rows before processing

3
GROUP BY

Organize data into groups

4
HAVING

Filter based on group stats

5
SELECT

Compute columns and expressions

6
DISTINCT

Remove any duplicate rows

7
ORDER BY

Sort prepared dataset

8
LIMIT

Final slice of the data
